RDF

RDF class final

Provides constants for the RDF vocabulary namespace (rdf:).

This namespace includes fundamental concepts of the RDF model like types, properties, list structures, reification, and datatypes.

See: https://www.w3.org/TR/rdf12-concepts/ See: https://www.w3.org/TR/rdf12-schema/
