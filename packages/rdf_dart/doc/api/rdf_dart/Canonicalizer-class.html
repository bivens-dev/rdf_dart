Canonicalizer

Canonicalizer class abstract base

Defines a standard way to create a unique, textual representation (canonical form) of an RDF dataset.

Think of it as a blueprint (or interface) for different algorithms that achieve the same goal: making sure the same dataset always produces the exact same text output, regardless of how its internal structure might vary. This is useful for comparing datasets or generating consistent signatures.

This class uses the Strategy Pattern, allowing different canonicalization algorithms like RDFC-1.0 or URDNA2015 to be used interchangeably.

Use the Canonicalizer.create factory to easily get an instance of a specific algorithm implementation.
