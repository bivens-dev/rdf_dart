IRINode

IRINode class

Represents an Internationalized Resource Identifier (IRI).

An IRI is a string that identifies a resource. This class ensures that the provided IRI string conforms to the basic IRI syntax rules. While this class performs basic validation, it doesn't check whether the resource referred to by the IRI actually exists.

This class is immutable. Once an IRI object is created, its value cannot be changed.
