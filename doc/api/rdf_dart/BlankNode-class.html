BlankNode

BlankNode class

Represents a blank node in an RDF graph.

Blank nodes are used to represent resources that do not have a globally unique identifier (IRI). They are locally scoped within the RDF graph and are often used for existential quantification or to represent unnamed or anonymous resources.

Blank nodes are identified by a unique string identifier, which can be provided by the user or automatically generated as a UUID. The identifier is only unique within the scope of the data it is defined, not globally.
