XSD

XSD class final

Provides constants for XML Schema Definition (XSD) datatype IRIs.

These IRIs are commonly used in RDF literals to specify the datatype of the literal's value.

See: https://www.w3.org/TR/xmlschema-2/ See: https://www.w3.org/TR/rdf12-concepts/#xsd-datatypes
