Creates a new Literal with the given lexicalForm, datatype, and optional language.

The lexicalForm is the string representation of the literal's value. The datatype is an IRI that specifies the type of the literal's value. The optional language is a language tag for text literals.

Example:

final myLiteral = Literal('example', IRI('http://www.w3.org/2001/XMLSchema#string'));
